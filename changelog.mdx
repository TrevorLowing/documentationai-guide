## Recent Updates

Stay informed about the latest enhancements to RMFCoach. We release updates regularly to align the simulation with evolving NIST SP 800-53 Rev 5 requirements and FedRAMP Moderate baselines.

<Callout kind="info">
Check back monthly for new scenarios and training tools that help you master real authorization decisions.
</Callout>

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>
## New Features

- Added three new cloud deployment scenarios covering hybrid environments with detailed Customer Responsibility Matrix exercises.
- Introduced team tabletop mode allowing up to six simultaneous roles including ISSO, AO, and CSP perspectives for collaborative training.

## Improvements

- Enhanced control narrative builder with real-time feedback on SSP quality based on actual FedRAMP review criteria.
- Updated the full set of AC, AU, IA, SC, and SI control families to reflect the latest NIST SP 800-53 Rev 5 revisions.

</Update>

<Update label="2024-09-20" description="v1.2.1" tags={["bugfix", "improvement"]}>
## Bug Fixes

- Fixed an issue where inheritance indicators failed to display correctly for shared CSP controls in the assessment phase.
- Resolved navigation errors in the security assessment workflow that occurred when switching between solo and team modes.

## Improvements

- Improved loading performance for large control sets so you can start practicing faster.
- Added clearer tooltips explaining the difference between inherited, shared, and customer-owned controls.

</Update>

<Update label="2024-08-10" description="v1.2.0" tags={["feature"]}>
## New Features

- Launched the complete three-phase authorization cycle simulation mirroring the full NIST RMF process from categorization through authorization decision.
- Added exportable reports summarizing your decisions for post-exercise review and team debriefs.

</Update>
